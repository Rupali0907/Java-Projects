# Java Quiz Game

A simple console-based Quiz Game built using Core Java.
The game asks multiple-choice questions with a timer and shows the final score and grade at the end.

## Features

* 10 Multiple Choice Questions
* Random question order
* 20-second timer for each question
* Score and percentage calculation
* Option to play again

## Technologies Used

* Java
* Arrays & ArrayList
* Collections (Shuffle)
* Timer & TimerTask
* Scanner

## How to Run

1. Download or clone the project.
2. Open it in IntelliJ IDEA or any Java IDE.
3. Run the `Main.java` file.
